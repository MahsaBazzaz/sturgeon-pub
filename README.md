# sturgeon

## Setup

To install pipenv, if needed:
```
pip3 install pipenv
```

To set up environment:
```
pipenv install
pipenv shell
```

See `examples.sh`, `examples-app.sh`, `examples-blend.sh`, `examples-mkiii.sh`, `examples-graph.sh` and `examples-explore.sh` for example usage (within pipenv).
