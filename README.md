# sturgeon

## Setup

To install pipenv, if needed:
```
pip3 install pipenv
```

To set up environment:
```
pipenv install
pipenv shell
```

See `examples.sh`, `examples-blend.sh`, `examples-mkiii.sh`, and `examples-graph.sh` for example usage (within pipenv).
